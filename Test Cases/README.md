# Test Cases — User Registration

## TC-001 — Valid registration
Steps:
1. Open registration page
2. Enter valid name
3. Enter valid email
4. Enter valid password
5. Click Register

Expected Result:
User account created successfully

---

## TC-002 — Invalid password
Steps:
1. Open registration page
2. Enter valid data
3. Enter password with less than 6 characters
4. Click Register

Expected Result:
Error message displayed
