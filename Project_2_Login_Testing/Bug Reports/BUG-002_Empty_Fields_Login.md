## Bug ID
BUG-002

## Title
Login allowed submission with empty mandatory fields

## Environment
- Browser: Google Chrome
- OS: Windows 10
- Environment: Test

## Steps to Reproduce
1. Access the login page
2. Leave email field empty
3. Leave password field empty
4. Click on the "Login" button

## Expected Result
The system should block submission and display validation messages for required fields.

## Actual Result
The system allows submission without showing validation feedback.

## Severity
High

## Status
Open
