## Bug ID
BUG-001

## Title
Incorrect error message displayed for invalid login credentials

## Environment
- Browser: Google Chrome
- OS: Windows 10
- Environment: Test

## Steps to Reproduce
1. Access the login page
2. Enter a valid email
3. Enter an incorrect password
4. Click on the "Login" button

## Expected Result
The system should display a clear error message indicating invalid credentials.

## Actual Result
The system displays a generic error message that does not guide the user.

## Severity
Medium

## Status
Open
