## Bug ID
BUG-003

## Title
Payment fails without displaying error message

## Environment
- Browser: Google Chrome
- OS: Windows 10
- Environment: Test

## Steps to Reproduce
1. Add a product to the cart
2. Proceed to checkout
3. Fill all mandatory fields
4. Enter invalid credit card data
5. Click on "Pay Now"

## Expected Result
System should block the payment and display a clear error message.

## Actual Result
Payment is not processed and no feedback message is shown to the user.

## Severity
High

## Status
Open
