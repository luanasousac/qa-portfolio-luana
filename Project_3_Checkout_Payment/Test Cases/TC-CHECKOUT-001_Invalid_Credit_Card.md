## Test Case ID
TC-CHECKOUT-001

## Title
Verify checkout behavior with invalid credit card data

## Preconditions
User has at least one product in the cart

## Test Steps
1. Navigate to checkout page
2. Fill billing and shipping information
3. Enter invalid credit card details
4. Click on Pay Now

## Expected Result
Payment should be rejected and an error message displayed

## Actual Result
Payment is rejected but no error message is shown

## Status
Failed
